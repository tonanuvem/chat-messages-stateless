# Chat com Websockets - Microservice Java Quarkus

Demonstração usando Quarkus: the Supersonic Subatomic Java Framework.

website: https://quarkus.io/

## Running the app

```
git clone
cd chat-messages-stateless
mv mvn .mvn
chmod +x mvnw
./mvnw quarkus:dev
```
